# NISSPY Frontend

React (Vite) uygulaması. Çalıştırmak için:

1. `npm install`
2. Bu klasördeki `.env.example` dosyasını `.env.local` olarak kopyalayıp (veya kökündeki `.env.example`'ı buraya `frontend/.env.local` olarak kopyalayıp) `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY` (ve isteğe bağlı `VITE_GEMINI_API_KEY`) doldurun.
3. `npm run dev` — http://localhost:3000

Docker kullanıyorsanız komutları proje kökünden çalıştırın: `docker compose up --build`.
