# Backend

Bu klasör ileride özel bir API sunucusu (örn. Node/Express) eklenmek üzere ayrıldı.

Şu an NISSPY uygulaması tamamen **frontend** tarafında çalışıyor; kimlik doğrulama ve veri Supabase, koç sohbeti ise Google Gemini API ile sağlanıyor. İleride sunucu taraflı işlemler gerekirse buraya eklenebilir.
